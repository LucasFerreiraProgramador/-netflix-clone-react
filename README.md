## 🎯 About

Clone da interface da Netflix desenvolvido com **React.js**, consumindo dados da API do **TheMovieDB (TMDB)**, com foco em componentização, requisições assíncronas e renderização dinâmica.

---

## 🚀 Features

- Listagem dinâmica de filmes  
- Banner principal com destaque  
- Scroll horizontal por categorias  
- Consumo da API TMDB  
- Layout inspirado na Netflix  
- Responsividade  

---

## 🧰 Technologies

- React.js  
- JavaScript  
- HTML5  
- CSS3  
- API TheMovieDB  

---

## 🧠 Estrutura do Projeto

```
src/

 ├── styles
```

Projeto estruturado com separação de responsabilidades e organização escalável.

## ✅ Requirements

Antes de começar, você precisa ter:

- Git  
- Node.js  

---

## 🏁 Starting

```bash
# Clone this project
git clone https://github.com/LucasFerreiraProgramador/netflix-clone-react

# Access
cd netflix-clone-react

# Install dependencies
npm install

# Run the project
npm start
```

O servidor iniciará em:

```
http://localhost:3000
```
